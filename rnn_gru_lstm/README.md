# nlp-foundations-to-llms:
=========================

## RNN, LSTM & GRU Architectures:
================================

## Overview:
===========

This directory explores the evolution of recurrent neural networks — from basic RNNs to advanced LSTM and GRU units. It provides hands‑on Jupyter notebooks, mathematical derivations, and visual diagrams to help learners and practitioners understand how gated architectures solve the vanishing gradient problem and enable long‑term sequence modeling.

## Contents:
===========
- RNN_basic.ipynb → Introduction to vanilla RNNs with text sequence examples.
- Mordern_RNN-LSTM_GRU.ipynb → Detailed implementations of LSTM and GRU units, including gate equations and training demos.
- Images (image-1.png … image-26.png) → Visual explanations of architectures, gates, and gradient flow.

## Key Features:
===============
- Step-by-step mathematical breakdowns of RNN, LSTM, and GRU update rules.
- Visual diagrams of gates, cell states, and hidden states.
- Hands‑on code for sequence prediction tasks.
- Comparison of RNN vs LSTM vs GRU in terms of performance, memory handling, and training stability.
- Recruiter‑friendly, modular notebooks designed for clarity and reproducibility.

## Learning Goals:
=================
- Understand why vanilla RNNs struggle with long‑term dependencies.
- Learn how LSTM gates (input, forget, output) and GRU gates (update, reset) solve gradient issues.
- Apply these models to NLP tasks such as next‑word prediction and sentiment analysis.
- Build a foundation for transitioning into Transformers and LLMs.
